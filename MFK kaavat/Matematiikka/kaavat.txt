

Matematiikka


Tämä osio sisältää keskeiset kaavat geometriasta, 
algebrasta, trigonometriasta ja analyysistä, 
joita tarvitaan esimerkiksi rakenteiden mitoituksessa, 
optimoinnissa ja tilastollisessa analyysissä.



Kaksiulotteiset muodot (2D)

◆ Neliö
  → Kuvaus: Sivun pituus s
  → Piiri: P = 4s
  → Pinta-ala: A = s²  
◆ Suorakulmio
  → Kuvaus: Sivujen pituudet a ja b
  → Piiri: P = 2a + 2b
  → Pinta-ala: A = ab  
◆ Ympyrä
  → Kuvaus: Säde r
  → Piiri: P = 2 * pi * r
  → Pinta-ala: A = pi * r²  
◆ Kolmio
  → Kuvaus: Sivut a, b, c, korkeus h
  → Piiri: P = a + b + c
  → Pinta-ala: A = (1/2)bh  
◆ Suunnikas
  → Kuvaus: Sivut a, b, korkeus h
  → Piiri: P = 2a + 2b
  → Pinta-ala: A = bh  
◆ Puolisuunnikas
  → Kuvaus: Sivut a, b, c, d, korkeus h
  → Piiri: P = a + b + c + d
  → Pinta-ala: A = (h/2)(a + b)  
◆ Ympyräsektori
  → Kuvaus: Säde r, kulma θ (asteina)
  → Ulkoreuna: L = pi * r * (θ/180°)
  → Pinta-ala: A = pi * r² * (θ/360°)  
◆ Ympyrän kaari
  → Kuvaus: Säde r, kulma θ (asteina)
  → Ulkoreuna: l = pi * r * (θ/180°)
  → Pinta-ala: –  
◆ Ympyrän rengas
  → Kuvaus: Ulkosäde R, sisäsäde r
  → Piiri: –
  → Pinta-ala: A = pi * (R² - r²)  
◆ Heronin kaava
  → Kuvaus: Sivut a, b, c; s = (a + b + c)/2
  → Piiri: –
  → Pinta-ala: A = √[s(s-a)(s-b)(s-c)]  
◆ Yleinen kolmio
  → Kuvaus: Sivut a, b, kulma C
  → Piiri: –
  → Pinta-ala: A = (1/2)ab * sin(C)  
◆ Säännöllinen n-kulmio
  → Kuvaus: n sivua, pituus s
  → Piiri: P = ns
  → Pinta-ala: A = (1/4)ns² * cot(pi/n)  


Kolmiulotteiset muodot (3D)

■ Kuutio  
Kuvaus: Sivun pituus l  

Pinta-ala: S = 6l²  

Tilavuus: V = l³

■ Suorakulmainen laatikko  
Kuvaus: Sivut a, b, c  

Pinta-ala: S = 2(ab + ac + bc)  

Tilavuus: V = abc

■ Lieriö  
Kuvaus: Säde r, korkeus h  

Pinta-ala: S = 2 * pi * r * (r + h)  

Tilavuus: V = pi * r² * h

■ Kartio  
Kuvaus: Säde r, korkeus h, sivujana s  

Pinta-ala: S = pi * r² + pi * r * s  

Tilavuus: V = (1/3) * pi * r² * h

■ Pallo  
Kuvaus: Säde r  

Pinta-ala: S = 4 * pi * r²  

Tilavuus: V = (4/3) * pi * r³

■ Yleinen pyramidi  
Kuvaus: Pohjan ala A, korkeus h  

Pinta-ala: –  

Tilavuus: V = (1/3) * A * h

■ Puolipallo  
Kuvaus: Säde r  

Pinta-ala: S = 3 * pi * r²  

Tilavuus: V = (2/3) * pi * r³

Pythagoraan lause
 Suorakulmaisessa kolmiossa: a² + b² = c²
 Esimerkki: Jos a = 3 ja b = 4, niin c = √(3² + 4²) = √25 = 5  

Eksponenttisäännöt
★ Eksponentti nolla
  · Sääntö: x^0 = 1, jos x ≠ 0
  · Esimerkki: 5^0 = 1  
★ Eksponentti yksi
  · Sääntö: x^1 = x
  · Esimerkki: 5^1 = 5  
★ Eksponenttien tulo
  · Sääntö: x^m · x^n = x^(m+n)
  · Esimerkki: 5^3 · 5^2 = 5^5  
★ Eksponenttien osamäärä
  · Sääntö: x^m / x^n = x^(m-n)
  · Esimerkki: 5^3 / 5^2 = 5  
★ Eksponentti potenssiin
  · Sääntö: (x^m)^n = x^(mn)
  · Esimerkki: (5^3)^2 = 5^6  
★ Tulon eksponentti
  · Sääntö: (xy)^m = x^m · y^m
  · Esimerkki: (5·2)^3 = 5^3·2^3  
★ Osamäärän eksponentti
  · Sääntö: (x/y)^m = x^m / y^m
  · Esimerkki: (5/4)^3 = 5^3 / 4^3  
★ Negatiivinen eksponentti
  · Sääntö: x^(-n) = 1 / x^n
  · Esimerkki: 2^(-1) = 1/2  
★ Murtoeksponentti
  · Sääntö: x^(a/b) = √[b]{x^a}
  · Esimerkki: 25^(1/2) = √25 = 5  


Algebra ja yhtälöt
 Lausekkeenkäsittely
  ‣ Sievennys, potenssi- ja kertolaskusäännöt, termien yhdistäminen.
  ‣ Esimerkki: 2x + 3x = 5x  
 Ensimmäisen asteen yhtälö
  ‣ ax + b = 0 ⟹ x = -b/a
  ‣ Esimerkki: 2x + 4 = 0 ⟹ x = -2  
 Toisen asteen yhtälö
  ‣ ax² + bx + c = 0 ⟹ x = (-b ± √(b² - 4ac)) / 2a
  ‣ Esimerkki: x² - 5x + 6 = 0 ⟹ x = 2 tai x = 3  
 Vastaavat epäyhtälöt
  ‣ Ratkaisu kuten yhtälössä, huom. suunnan vaihto negatiivisella kertoimella.
  ‣ Esimerkki: -2x > 4 ⟹ x < -2  
 Juuriyhtälö
  ‣ √x = a ⟹ x = a²
  ‣ Esimerkki: √x = 3 ⟹ x = 9  
 Lineaarinen yhtälöpari
  ‣ Esim. { ax + by = c, dx + ey = f }, ratkaisu sijoittamalla tai yhteenlaskumenetelmällä.
  ‣ Esimerkki: { 2x + y = 5, x - y = 1 } ⟹ x = 2, y = 1  
 Yhtälöryhmä
  ‣ Useampi kuin 2 yhtälöä, ratkaisu esim. Gaussin eliminointi.  



Geometria ja trigonometria
✦ Suorakulmaisen kolmion ratkaisu
  ⇒ Pythagoraan lause: a² + b² = c²
  ⇒ Trigonometriset funktiot: sin, cos, tan
  ⇒ Esimerkki: sin(θ) = vastainen/hypotenusa  
✦ Yleinen kolmio
  ⇒ Sinilause: a/sin(A) = b/sin(B) = c/sin(C)
  ⇒ Kosinilause: c² = a² + b² - 2ab * cos(C)  
✦ Trigonometriset funktiot
  ⇒ sin x, cos x, tan x, yksikköympyrä.
  ⇒ Arvot: sin(0) = 0, cos(0) = 1, tan(0) = 0  
✦ Trigonometriset kaavat
  ⇒ sin(2x) = 2 * sin(x) * cos(x)
  ⇒ cos(2x) = cos²(x) - sin²(x)
  ⇒ cos²(x) + sin²(x) = 1  
✦ Trigonometriset yhtälöt
  ⇒ Ratkaistaan yksikköympyrällä ja perusratkaisuilla.
  ⇒ Esimerkki: sin(x) = 0 ⟹ x = n * pi  



Funktiot
▼ Funktiokäsite
  └ Määrittelyjoukko, arvojoukko, lauseke.
  └ Esimerkki: f(x) = x², määrittelyjoukko ℝ  
▼ Polynomifunktiot
  └ Ensimmäisen asteen: f(x) = ax + b
  └ Toisen asteen: f(x) = ax² + bx + c  
▼ Eksponenttifunktio
  └ f(x) = a^x, a > 0, a ≠ 1
  └ Esimerkki: f(x) = 2^x  
▼ Logaritmifunktio
  └ log_a(x), käänteisfunktio eksponentille.
  └ Esimerkki: log_2(8) = 3  
▼ Logaritmin laskusäännöt
  └ log_a(xy) = log_a(x) + log_a(y)
  └ log_a(x/y) = log_a(x) - log_a(y)
  └ log_a(x^b) = b * log_a(x)  
▼ Logaritminen asteikko
  └ Arvot kasvavat eksponentiaalisesti, esim. desibelit.
  └ Esimerkki: dB = 10 * log_10(P/P_0)  


Analyysi
◇ Raja-arvo
  ✧ lim(x→a) f(x)
  ✧ Esimerkki: lim(x→2) (x² - 4)/(x - 2) = 4  
◇ Derivaatan määritelmä
  ✧ f'(x) = lim(h→0) (f(x+h) - f(x))/h  
◇ Derivointisäännöt
  ✧ Summa: (f + g)' = f' + g'
  ✧ Tulo: (fg)' = f'g + fg'
  ✧ Ketjusääntö: (f(g(x)))' = f'(g(x)) * g'(x)
  ✧ Vakiot: (c)' = 0
  ✧ Potenssi: (x^n)' = n * x^(n-1)
  ✧ Eksponentti: (e^x)' = e^x
  ✧ Logaritmi: (ln x)' = 1/x
  ✧ Trigonometriset: (sin x)' = cos x, (cos x)' = -sin x  
◇ Ääriarvot
  ✧ Paikalliset maksimit/minimit: f'(x) = 0
  ✧ Esimerkki: f(x) = x³ - 3x, f'(x) = 3x² - 3 = 0 ⟹ x = ±1  
◇ Osittaisderivaatta
  ✧ ∂f/∂x, ∂f/∂y
  ✧ Esimerkki: f(x,y) = x² + y², ∂f/∂x = 2x  
◇ Virhearviointi
  ✧ Δf ≈ f'(x)Δx
  ✧ Esimerkki: f(x) = x², x = 2, Δx = 0.1, Δf ≈ 4 * 0.1 = 0.4  
◇ Määrätty integraali
  ✧ ∫_a^b f(x) dx
  ✧ Esimerkki: ∫_0^1 x² dx = [x³/3]_0^1 = 1/3  
◇ Integrointisäännöt
  ✧ Perusfunktiot: ∫x^n dx = (x^(n+1))/(n+1) (n ≠ -1)
  ✧ ∫1/x dx = ln|x| + C
  ✧ ∫e^x dx = e^x + C
  ✧ ∫sin(x) dx = -cos(x) + C
  ✧ ∫cos(x) dx = sin(x) + C
  ✧ Osittaisintegrointi: ∫u dv = uv - ∫v du  
◇ Integraalifunktio
  ✧ F(x) = ∫ f(x) dx
  ✧ Esimerkki: F(x) = ∫x² dx = x³/3 + C  
◇ Sovelluksia
  ✧ Pinta-alat, tilavuudet, keskiarvot.
  ✧ Esimerkki: Keskiarvo = (1/(b-a))∫_a^b f(x) dx  
◇ Separoituva diff.yhtälö
  ✧ dy/dx = g(x)h(y) ⇒ ∫(1/h(y)) dy = ∫g(x) dx  
◇ 1. kertaluvun lineaarinen
  ✧ dy/dx + P(x)y = Q(x)
  ✧ Ratkaisu: y = e^(-∫P(x)dx) (∫Q(x)e^(∫P(x)dx) dx + C)  
◇ 2. kertaluvun vakiokertoiminen
  ✧ ay'' + by' + cy = 0
  ✧ Ratkaisu: y = e^(rt), missä r² + (b/a)r + c/a = 0 


Todennäköisyys ja tilastot
▷ Kombinatoriikka
  ↳ Permutaatiot: n!
  ↳ Yhdistelmät: C(n, k) = n!/(k!(n-k)!)  
▷ Klassinen todennäköisyys
  ↳ P = suotuisat / mahdolliset
  ↳ Esimerkki: Nopan heitto, P(6) = 1/6  
▷ Ehdollinen todennäköisyys
  ↳ P(A|B) = P(A ∩ B) / P(B)
  ↳ Esimerkki: P(A|B) = 0.3/0.5 = 0.6  
▷ Satunnaismuuttuja
  ↳ Diskreetti ja jatkuva, esim. noppa ja normaalijakauma.  
▷ Tunnusluvut
  ↳ Odotusarvo: E(X) = Σx_i * P(x_i) (diskreetti)
  ↳ Varianssi: Var(X) = E(X²) - [E(X)]²
  ↳ Keskihajonta: √Var(X)  
▷ Binomijakauma
  ↳ P(X = k) = C(n, k)p^k(1-p)^(n-k)
  ↳ Esimerkki: n = 5, p = 0.5, P(X = 3)  
▷ Poisson-jakauma
  ↳ P(X = k) = (λ^k e^(-λ)) / k!
  ↳ Esimerkki: λ = 2, P(X = 1) = 2e^(-2)/1!  
▷ Normaalijakauma
  ↳ N(μ, σ²), Z = (X - μ)/σ
  ↳ Esimerkki: N(0,1), P(Z < 1) ≈ 0.8413  
▷ Eksponenttijakauma
  ↳ f(x) = λ e^(-λx), x ≥ 0
  ↳ Esimerkki: λ = 1, P(X < 1) = 1 - e^(-1)  
▷ Parametrien estimointi
  ↳ Otoskeskiarvo: x̄ = Σx_i/n
  ↳ Otoshajonta: s = √[Σ(x_i - x̄)²/(n-1)]  
▷ Hypoteesien testaus
  ↳ H₀, testisuure, p-arvo.
  ↳ Esimerkki: p < 0.05 ⟹ hylätään H₀  
▷ Empiirinen aineisto
  ↳ Mediaani, moodi, keskiarvo, hajonta.
  ↳ Esimerkki: Data: 1,2,3,4,5, keskiarvo = 3
