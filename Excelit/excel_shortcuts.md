# Excel-pikanäppäimet insinööreille

## Perusnäppäimet
- `Ctrl + S`: Tallenna tiedosto.
- `Ctrl + C` / `Ctrl + V`: Kopioi/Liitä.
- `Ctrl + Z`: Kumoa.
- `F2`: Muokkaa aktiivista solua.
- `Alt + =`: Automaattinen summa (lisää `=SUMMA()`).

## Datan käsittely
- `Ctrl + Shift + L`: Lisää suodatin.
- `Alt + A + S + S`: Lajittele A–Z.
- `Alt + A + S + Z`: Lajittele Z–A.
- `Ctrl + Shift + $`: Muotoile valuuttamuotoon.
- `Ctrl + Shift + %`: Muotoile prosenttimuotoon.

## Kaaviot ja muotoilu
- `F11`: Luo kaavio uudelle välilehdelle.
- `Ctrl + 1`: Avaa solujen muotoiluvalikko.
- `Alt + H + B + T`: Lisää paksu reunaviiva.
- `Alt + H + A + C`: Keskitä teksti.

## Siirtyminen ja valinta
- `Ctrl + Page Up/Down`: Siirry välilehtien välillä.
- `Ctrl + Shift + Arrow`: Valitse alue nopeasti.
- `Ctrl + G`: Siirry tiettyyn soluun (esim. A100).

## VBA ja kehittyneet
- `Alt + F11`: Avaa VBA-editori.
- `Alt + F8`: Näytä makrot.
- `Alt + Q`: Sulje VBA-editori.

## Vinkkejä
- Käytä `F4` toistaaksesi viimeisen toiminnon (esim. muotoilu).
- `Ctrl + D`: Kopioi arvo yläpuolelta.
- `Ctrl + R`: Kopioi arvo vasemmalta.
- Ota käyttöön pikanäppäimet: File → Options → Quick Access Toolbar.
