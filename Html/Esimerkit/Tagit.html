HTML Tagit - Esimerkit

1. Perusrakenne

<html>: Määrittää HTML-dokumentin juurielementin. Kaikki muut tagit ovat tämän sisällä.

<head>: Sisältää metatietoja (otsikko, CSS, JavaScript), jotka eivät näy suoraan käyttäjälle.

<title>: Määrittää sivun otsikon, joka näkyy selaimen välilehdessä.

<body>: Sisältää sivun näkyvän sisällön (teksti, kuvat, linkit jne.).

Esimerkki:

<html lang="en">
    <head>
        <meta charset="UTF-8">
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <title>Oma verkkosivuni</title>
    </head>
    <body>
        <h1>Hei maailma!</h1>
        <p>Tämä on minun ensimmäinen verkkosivuni.</p>
    </body>
</html>

2. Dokumentin metatiedot

<base>: Määrittää perus-URL-osoitteen sisäisille linkeille.

<meta>: Antaa metatietoja (kuvaus, avainsanat, Open Graph -tagit).

<link>: Linkittää ulkoisia resursseja, kuten CSS-tiedostoja.

<style>: Sisältää sisäisiä CSS-tyylejä.

Esimerkki:

<html lang="en">
    <head>
        <base href="http://myfirstwebsite.com" target="_blank">
        <meta charset="UTF-8">
        <meta name="description" content="Tämä on minun sivuni">
        <meta name="keywords" content="HTML, CSS, JavaScript">
        <meta property="og:title" content="Minun kaunis sivuni">
        <meta property="og:description" content="Tervetuloa sivulleni!">
        <title>Minun kaunis sivuni</title>
        <link rel="stylesheet" href="/css/master.css">
        <style>
            body { font-family: Arial, sans-serif; background-color: #f0f0f0; }
            h1 { color: #333; }
        </style>
    </head>
    <body>
        <h1>Otsikko</h1>
        <p>Tässä on <a href="/about">linkki</a> meistä-sivulle.</p>
    </body>
</html>

5. Kuvat

Kuvien käsittely HTML:ssä sisältää peruskuvat, klikattavat alueet, responsiiviset kuvat ja optimoinnin.

5.1 Peruskuvat

<img>: Näyttää kuvan. Pakolliset attribuutit: src, alt.

Attribuutit:

  • src="...": Kuvan polku tai URL.
  • alt="...": Vaihtoehtoinen teksti saavutettavuutta varten.
  • height="...": Korkeus (px tai %).
  • width="...": Leveys (px tai %).
  • loading="lazy": Lataa kuva vasta, kun se tulee näkyviin (optimoi suorituskykyä).

5.2 Klikattavat kuva-alueet

<map> ja <area>: Mahdollistavat klikattavat alueet kuvassa.

Attribuutit:

  • usemap="#mapname": Linkittää kuvan karttaan (käytetään <img>-tagissa).
  • shape="rect|circle|poly": Alueen muoto.
  • coords="...": Alueen koordinaatit (esim. suorakulmiolle: x1,y1,x2,y2).
  • href="...": Linkki, johon alue ohjaa.

5.3 Responsiiviset kuvat

<picture> ja <source>: Mahdollistavat eri kuvien lataamisen näytön koon tai laitteen mukaan.

Attribuutit:

  • media="...": Media query, joka määrittää, milloin lähdekuvaa käytetään.
  • srcset="...": Kuvan polku.
